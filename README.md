# AI Mock Interview Project

The given project is a website created for developers to test their skills by taking mock interviews with AI. This helps them understand where they stand in terms of their skills and which topics they need to revise or study in order to ace interviews.

## TechStack 
- Java
- Spring Boot 
- React (Basically my personal practice instead of HTML , CSS, Javascript)
- MySQL for the given Data Base 

## Features 
- Login Page(SignIn Register).
- Analytical Page for a user based on his skills. 
- Chat Bot AI  b/w user and interviewer (where interviewer is a chat bot).
- Time && Difficulty Selection of user for the interviewer.
- Experience
